# TheAdventuresOf
Short game about The Adventures Of... Simple 2d sidescroller (sort-of)

## Running in 2026 (macOS Desktop)

This repo now includes a DesktopGL launcher that works with modern .NET + MonoGame:

```bash
dotnet run --project DesktopGL/TheAdventuresOf.DesktopGL.csproj
```

Notes:
- The first run restores the local `dotnet-mgcb` tool from `.config/dotnet-tools.json`.
- Content is compiled from `Android/Assets/Content/Content.mgcb` during build.
